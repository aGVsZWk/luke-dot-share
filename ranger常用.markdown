`zh`或`<C-h>`: 显示隐藏文件   
`shift+j/k`: 跳到第一行/最后一行   
`ctrl+f`: 搜索, 调用安装的工具   
`dD`: 删除   
`dd`: 剪切   
`yy`: 复制   
`pp`: 粘贴   
`/`: 搜索, 按Tab可跳过去   
`[`和`]`: 父级移动   
`I`: 选择打开方式   
`i`: 文件名前插入   
`a`: 文件名后，格式之前追加   
`A`: 文件名格式之后追加   
`cw`: 重命名   
`Space`: 选中   
`v`: 全选   
`f`: 当前过滤至唯一并直接进入   
`<S-v>`: shell vim 新建文件   
`Ins`: 新建文件   
`Del`: 删除文件   
`yp`: 复制绝对路径   
`yn`: 复制名称   
`dU`: 显示文件夹大小   
`gi`: 跳到家目录下Github   
`gD`: 跳到家目录下Download
